# 🛠️ Manual do Síndico — Reserva da Quadra (Playa del Mago)
**Acesse:** https://playa-quadra.vercel.app · entre com seu **CPF + senha** de administrador.
Só o administrador vê o painel **Admin** (botão no topo).

---

## 1. Primeiro passo: cadastrar os moradores
No painel, em **Importar moradores (em massa)**, cole a lista (uma linha por morador):
```
CPF, Nome, Unidade, E-mail(opcional)
123.456.789-09, Ana Lima, Bloco A 101, ana@email.com
```
- **Deixe a senha em branco** → cada morador cria a própria senha sozinho (recomendado).
- Avise os moradores para entrarem em **"Primeiro acesso? Ative sua conta"** (CPF + unidade).

> Também dá para cadastrar **um a um** logo abaixo.

## 2. Regras da quadra (sem programador)
Em **Regras**, você ajusta e salva:
- Horário de **abertura/fechamento** e antecedência de reserva.
- **Limite** de reservas ativas e por semana, por morador.
- Tempo mínimo de **cancelamento** e **dias de bloqueio por falta**.

## 3. Ações do dia a dia
| Quero… | Onde |
|---|---|
| Ver uso, faltas e ranking | **Relatório** (topo do painel) |
| Bloquear a quadra (manutenção, torneio, chuva) | **Bloquear a quadra** (data + horário) |
| Criar reserva fixa (ex.: aula toda terça 19h) | **Reserva fixa semanal** |
| Marcar quem faltou | Botão **falta** no morador → bloqueia X dias |
| Desbloquear / perdoar | **Desbloquear** no morador |
| Redefinir senha de um morador | Botão **"Gerar"** → mostra a senha na tela |
| Suspender / reativar acesso | Botões na lista de moradores |

## 4. Dicas
- O sistema **impede** dois moradores no mesmo horário automaticamente.
- Quem **falta sem cancelar** é bloqueado pelos dias que você definir.
- Tudo roda na nuvem — **nada para instalar** no condomínio.
